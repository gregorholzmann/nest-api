# nest-api

## Description

A simple API built with Nest.js and GraphQL
Run along with [example UI app](https://github.com/gregorholzmann/nest-api-ui) and a local instance of [MongoDB](https://www.mongodb.com/) to interact with the API.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
